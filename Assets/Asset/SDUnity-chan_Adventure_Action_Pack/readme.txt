SD Unity-Chan Adventure Action Pack / SD 유니티짱 어드밴쳐 액션 팩

캐릭터 라이센스 : unity-chan.com
제작자 : studio haon (http://rugiadian.wix.com/Haon)

이 어셋은‘유니티 테크놀로지 재팬’에서 공식으로 제공되는, <SD 유니티짱 3D모델 데이터> 패키지에 호환되는 모션 파일팩입니다. 


- 공식 <SD유니티짱 3D 모델 데이터>를 함께 사용하시길 추천합니다. 

이 모션들을 유니티짱이 아닌 모델에 적용해도 문제없습니다. 다만 공식적으로 제공되는 SD유니티짱 모델 데이터를 사용할 것을 가정로 만들어진 모션팩입니다. 
그래서 공식 모션데이터에 포함있는 모션과 용도가 겹칠 것이라 여겨진 것은 제작하지 않았습니다. 

예를들어 공식 패키지에 걷기와 달리기가 있기 때문에 이 패키지에는 걷기와 달리기가 없습니다. 스탠딩과 점프는 공식소스에 있지만 다른 스타일로 만들어 넣었습니다. 


-모션 이름 규칙에 예외사항이 있습니다.
기본적으로 같은 테마의 모션을 하나의 fbx파일에 넣고, 각 모션의 이름을 '주제_부분@옵션'의 규칙으로 정의 되어있습니다만 
사용자가 공식 패키지와 이 패키지를 함께 사용한다는 가정하에 예외를 만들었습니다. 
GoDown_Back.fbx에 포함된 모션들의 이름은 공식 유니티짱의 GoDown,DownHold,DownToUp과 유사하게 정의되어 있습니다. 


- 스튜디오 홈페이지에서 “어셋 업그레이드”로 추가되길 희망하는 모션을 요청받고 있습니다.

This Asset is a motion file package compatible with <SD Unity-chan 3D Model Data> officially provided  by Unity Technologies Japan.


- We recommend you using it with official <SD Unity-chan 3D Model Data>.

Althogh these motions can be adapted to other 3D models than SD Unity-chan, they are basically intended to be used with official <SD Unity-chan 3D Model Data>.
So the motions already included in the official package are not presented in this pack.
For example, running and walking motion are in the official package, therefore not in this package. There are standing & jumpping motions in both packages but differs in style.



-There’s an exception on our naming system.

Basically we put multiple motions sharing the same theme in one fbx file, and named each motion 'Theme_part@option' accordingly. 
The exception has been made assuming that our user will use this package together with the official package.
The names of the motions included in GoDown_Back.fbx are similar to <GoDown>, <DownHold>, <DownToUp> motions in the official Unity-chan.


- Please feel free to requests for additional motions through our studio’s website.
